# 🤖 Generative AI ChatBot

A modern AI-powered chatbot built using **Streamlit**, **LangChain**, and **Groq API** with a beautiful UI and real-time streaming responses.

---

## 🚀 Features

- ✅ Modern Dark UI
- ✅ Multiple AI Model Support
- ✅ Streaming Responses
- ✅ Chat Memory
- ✅ Sidebar Settings
- ✅ Temperature Control
- ✅ Fast AI Responses using Groq
- ✅ Built with Streamlit + LangChain

---

## 🛠️ Technologies Used

- Python
- Streamlit
- LangChain
- Groq API
- dotenv

## ⚙️ Installation

### 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/generative-ai-chatbot.git
```

### 2️⃣ Move into Project Folder

```bash
cd generative-ai-chatbot
```

### 3️⃣ Create Virtual Environment

```bash
python -m venv .venv
```

### 4️⃣ Activate Virtual Environment

#### Windows

```bash
.venv\Scripts\activate
```

#### Mac/Linux

```bash
source .venv/bin/activate
```

---

## 📦 Install Requirements

```bash
pip install -r requirements.txt
```

---

## 🔑 Setup Environment Variables

Create a `.env` file in the root folder.

```env
GROQ_API_KEY=your_groq_api_key
```

---

## ▶️ Run the Application

```bash
streamlit run chatbot.py
```
